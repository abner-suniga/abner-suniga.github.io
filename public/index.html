My story

I was born in Maringá, Brazil, in 1997. As a kid, I loved playing and watching others play video games. I excelled in math and physics, so I knew I would study something related to computers in college.

Starting Computer Science, I knew almost nothing about programming, except for a bit of HTML, CSS, and JavaScript to create a basic web page. It's funny that we only had one programming class during the second semester because the first two semesters were packed with math courses. At the time, it was hard to see how the small programs we made would build up to large applications. Even today, it's still challenging for me to understand how everything works, from the tiny transistors in a computer to the pixels you see on the screen. We build on top of a giant, messy amount of code that has been developed over years.

In my second year, I started learning web development. I chose the web path because I wanted to build something, an app, whatever, so I could make millions! This hasn't happened yet. However, throughout my career, I have built a lot of applications with the classic stack of React, Node.js, and PostgreSQL, sometimes using other tools like Redis, messaging queue systems like RabbitMQ, and many libraries thanks to NPM.

I've developed software for some traditional companies in the past, but my focus has always been on startups, where I find the best opportunities to learn and explore new ideas. My goal with this blog is to explain what I know about tech and life in a simple yet profound way. I hope you find my posts useful and that they contribute to your development as a builder.
